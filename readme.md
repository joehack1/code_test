CSS Match Challenge 🎯
A precision-based HTML/CSS comparison game that tests your frontend skills by matching target outputs pixel-perfect.

🚀 Features
Code Comparison: Write HTML/CSS to match target designs with accuracy scoring

Real-time Preview: See your output live alongside the target

Visual Diff: Detailed breakdown of differences between your code and target

Dark Theme: Smooth, eye-friendly interface with gradient accents

Progress Tracking: Track your accuracy, level, and score

Multiple Challenges: Ranging from beginner to expert difficulty

🛠️ Tech Stack
Frontend: HTML5, CSS3, Vanilla JavaScript

UI Framework: Custom CSS with CSS Grid & Flexbox

Icons: Font Awesome 6

Editor: Monaco Editor (to be integrated)

Comparison Engine: HTML/CSS parsing libraries

📋 Prerequisites
Modern web browser (Chrome, Firefox, Safari, Edge)

Basic HTML/CSS knowledge

No installation required - runs directly in browser

🎮 How to Play
View Target: Examine the target output in the left panel

Write Code: Type your HTML/CSS in the code editor

Test Solution: Click "Test My Code" to compare

Improve: Review differences and refine your code

Advance: Achieve 90%+ accuracy to unlock next challenge

🎯 Challenge Types
Beginner: Basic HTML elements and CSS properties

Intermediate: Layouts, positioning, and responsive design

Advanced: Animations, complex layouts, and CSS Grid/Flexbox

Expert: Pixel-perfect recreations of real-world components

🏆 Scoring System
Accuracy: Percentage match to target output

Speed Bonus: Extra points for faster completion

Efficiency: Points for cleaner, more optimized code

Streak Multiplier: Consecutive successful challenges
